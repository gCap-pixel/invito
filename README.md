# 🏎️ OPERAZIONE: CENA DI FINE ANNO (The Prof Championship)

Benvenuta Prof! Questo è il solito progetto Java con mille bug. 

L'obiettivo? Portarla sana e salva al traguardo più importante dell'anno: la **Cena di Fine Anno**! 🍾

## 🛠️ Requisiti di Sistema (e di Sopravvivenza)

* **Java** (Versione 8 o superiore... o qualsiasi versione che non faccia esplodere il PC)
* **Pazienza** (La stessa che ha avuto con noi durante tutto l'anno scolastico)
* **Appetito**

## 🎮 Regole del Gioco

* **Se vince la Prof**: Ha l'obbligo morale di venire alla cena per festeggiare sul podio.
* **Il giorno e l'ora** verranno rivelati solo se vincerà la gara.
